# Login System Demo

This is a full-stack login system built with a React frontend and backend APIs.

## Features
- Email and password login
- Frontend form validation
- Backend authentication APIs
- Secure API structure
- Clean and responsive UI

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MySQL (or MongoDB / PostgreSQL)
- Styling: CSS / SCSS / Tailwind (as used)

## Setup Instructions

### Frontend
```bash
cd frontend
npm install
npm start

cd backend
npm install
npm start
